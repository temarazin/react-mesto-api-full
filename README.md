# react-mesto-api-full

[Live App](https://temarazin.nomoredomains.xyz/)

**Описание**

Сервис Mesto: интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.

**Используемые технологии**

- Семантика HTML5
- БЭМ (nested)
- Flexbox и Grid
- Media queries
- Javascipt
- React

Api:
- Node.js
- Express.js
- MongoDB
